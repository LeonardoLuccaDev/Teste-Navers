O repositório encontra-se tanto a lista de exercicios quanto a API do teste pratico a pedido da empresa Navers.

INSTRUÇÕES PARA A API:

-Abra o terminal e vá até entrar na pasta API Naver.
-Instale as dependências (express sqlite3 typeorm @types/express ts-node-dev typescript).
-Para iniciar o projeto digite yarn dev.

ROTAS:

GET http://localhost:3333/navers
GET http://localhost:3333/navers/:id
GET http://localhost:3333/projects
GET http://localhost:3333/projects/:id
POST http://localhost:3333/navers
POST http://localhost:3333/projects
